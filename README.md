# Housing Society Management
#### CakePHP version 3.8

## Features
1. Residents Records
  * Name
  * Email
  * Phone
  * Type (Owner/Tenant)
  * Flat No.

2. Visitor Log
  * Name
  * Phone
  * Purpose
  * In Time
  * Out Time

3. Events
  * Name
  * Date & Time
  * Location

4. Complaint Board
  * Complaint
  * Date & Time
  * Status(Pending/Resolved)

## Other Common Functionalities
  * Add Records
  * Edit Records
  * Delete Records
